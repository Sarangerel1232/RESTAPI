# Music REST API

Simple Spring Boot REST API for managing singers, albums, and songs.

## Tech
- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Maven

## Features
- CRUD for Singer
- Album linked to Singer
- Song linked to Album and Singer
- RESTful endpoints

## Run
mvn spring-boot:run

## URLs
App: http://localhost:8080  
H2 Console: http://localhost:8080/h2-console
